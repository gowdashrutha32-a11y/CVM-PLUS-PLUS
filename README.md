# CVM++

A custom scripting language and stack-based virtual machine developed in C++.

CVM++ demonstrates the complete workflow of a programming language implementation, beginning from source code and ending with execution on a custom virtual machine.

The project includes:

- Lexical Analysis (Lexer)
- Syntax Analysis (Parser)
- Abstract Syntax Tree (AST)
- Bytecode Compilation
- Stack-Based Virtual Machine
- Script Execution from `.cvm` files

# Features

## Lexer

The lexer scans source code and converts it into tokens.

Example:

```c
var x = 10;
```

Generated Tokens:

```txt
VAR
IDENTIFIER
ASSIGN
NUMBER
SEMICOLON
```

---

## Parser

The parser converts tokens into an Abstract Syntax Tree (AST).

Example AST:

```txt
VAR_DECL -> x=10
PRINT -> x
ASSIGN -> x=50
```

---

## Bytecode Compiler

The compiler converts AST nodes into low-level bytecode instructions understood by the virtual machine.

Example:

```txt
PUSH 10
STORE x

LOAD x
PRINT
```

---

## Virtual Machine

The virtual machine executes bytecode instructions using a stack-based execution model.

Supported Instructions:

```txt
PUSH
STORE
LOAD
ADD
PRINT
```

# Build Instructions

Compile:

```bash
g++ src/main.cpp src/lexer.cpp src/parser.cpp src/compiler.cpp src/vm.cpp -o cvm
```

Run:

```bash
.\cvm.exe
```

# Execution Stages

When the program is executed, CVM++ displays:

1. Generated Tokens
2. Generated AST
3. Generated Bytecode
4. Final Program Output

This allows easy debugging and visualization of every stage of the compilation process.
